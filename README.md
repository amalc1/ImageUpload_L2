# image-upload

single image upload

### clone Project

Clone the project

```bash
  git clone https://github.com/amalc1/ImageUpload_L2.git
```

### Run Locally

Go to the project directory

```bash
  cd ImageUpload_L2
```

navigate to the client(react)

```bash
  cd client
```

Install dependencies

```bash
  npm install
cd ..
```

navigate to the server(node-express)

```bash
  cd server
```

Install dependencies

```bash
  npm install
  cd ..
```

### Start the front-End React subfolder

```bash
cd client
npm run start
```

### Start the server subFolder

```bash
cd ..
cd server
npm run start
```

### Go to home page

```http
    GET /locahost:3000
```
